# The Machine - Demo

Frontend-only demo version of The Machine.

## Quick Start

\\\ash
npm install
npm run dev
\\\

## Deploy to Vercel

1. Push to GitHub
2. Import to Vercel: https://vercel.com/
3. Vercel will auto-detect Vite and deploy

Or use CLI:
\\\ash
npm install -g vercel
vercel
\\\
